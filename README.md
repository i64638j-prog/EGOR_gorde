## Описание программы
Movie Library - это графическое приложение на Python для ведения личной кинотеки. Программа позволяет:

- Добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга
- Просматривать все фильмы в удобной табличной форме
- Фильтровать фильмы по жанру и году выпуска
- Просматривать статистику (общее количество, средний/лучший/худший рейтинг)
- Удалять фильмы двойным кликом
- Сохранять и загружать библиотеку в JSON-формате
- Проверять корректность вводимых данных

## Требования к системе

### Программные требования
- Python 3.6 или выше
- Библиотеки: только стандартные (tkinter, json)

### Установка Python
- **Windows**: Скачайте с [python.org](https://www.python.org/downloads/)
- **macOS**: `brew install python3`
- **Linux**: `sudo apt-get install python3`

### Проверка tkinter

**Windows/Mac**: Tkinter обычно установлен по умолчанию

**Linux**:
```bash
sudo apt-get install python3-tk  # Ubuntu/Debian
